# 使用 C++，基于 Drogon 后端+React 前端开发一个简单的模拟金融系统

## 简介

本项目旨在使用 Drogon 作为后端框架，使用 C++语言进行开发，并利用 React 作为前端框架，实现一个简单的模拟金融系统。该系统的主要功能包括账户的转账、取款、存款、利息结算、交易记录等，同时前端提供优美的界面，并支持暗黑模式切换。后端使用 ORM 框架和数据库对接，以保证系统的高效性和数据安全性。

## 任务列表

- 学习 Drogon 框架和 ORM 的使用，了解 Restful API 的开发和使用，并与数据库对接。
- 根据需求，设计并实现 Restful API 接口，包括但不限于账户的转账、取钱、存钱、利息结算和交易记录等功能。
- 使用 C++语言编写后端代码，包括但不限于业务逻辑、验证逻辑和数据模型等。
- 编写数据库表结构，使用 ORM 框架进行数据库操作，包括但不限于数据查询、插入、更新和删除操作。
- 进行单元测试，确保代码质量和系统健壮性。
- 学习 React 框架和前端页面的开发和使用。
- 根据设计，开发并测试前端页面，包括但不限于账户的转账、取款、存款、利息结算和交易记录等功能。
- 提供暗黑模式，兼容性和 ui 交互美观
- 将前端和后端进行整合测试，确保系统的基本功能正常运行。
- 进行性能测试和压力测试，优化代码以提高系统性能和并发能力。

## 技术路线

### 后端

- 使用 Drogon 框架搭建后端，其中包含 Web 服务器、Http 客户端以及 Restful API 的支持。
- 使用 SQLite 作为数据库，它是一个轻量级的关系型数据库管理系统，容易安装和部署，适合小型 Web 应用。
- 使用 ORM 框架进行数据库操作

### 前端

- 使用 React 作为前端框架，利用 mobX 进行状态管理，以实现页面的渲染和交互。
- 使用 joy-ui UI 框架进行页面的设计和美化，同时加入暗黑模式支持。
- 使用 axios 从后端获取数据，并在前端进行显示，实现账户的转账、取钱、存钱、利息结算和交易记录等功能。
